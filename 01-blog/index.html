从 0 搭建 Hexo + Fluid + GitHub Pages 博客

本文是一个从零开始搭建 Hexo 博客的教程，使用 Fluid 主题并部署到 GitHub Pages。适合初学者和想要快速搭建个人博客的人。➡️

1. 整体思路（先有个全局概念）

整体来说，需要做以下几件事：

  1. 安装环境：Node.js、Git

  2. 用 Hexo 生成一个本地博客

  3. 把主题换成 Fluid

  4. 写几篇文章、关于页

  5. 配好 GitHub Pages，一条命令部署到网上
